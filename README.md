## Welcome to Object-Oriented Programming

"Object-Oriented Programming, or "OOP", is a pattern for (allegedly) writing clean and maintainable code. Admittedly, not everyone agrees that object-oriented programming is the best way to write code, but, to be a good engineer, you should at least understand it." --bootdev-author

- feat – new user-visible feature: `feat: add pagination to /books endpoint-prod-[API-123]`

- fix – bug fix: `fix: handle null user_id in session lookup-sentry-[BUG-456]`

- chore – non-feature, non-bug housekeeping: `chore: bump FastAPI to 0.115.0-deps-[OPS-022]`

- perf – performance improvement: `perf: cache user profile lookups with Redis-redis-[PERF-012]`

- test – add or improve tests: `test: add integration tests for checkout flow-ci-[TEST-088]`
